Создание проекта

Использование vue-cli

Рекомендуем создавать проекты, использующие vue-loader, с помощью vue-cli:

npm install -g vue-cli
vue init webpack-simple hello-vue
cd hello-vue
npm install
npm run dev # можно начинать!
